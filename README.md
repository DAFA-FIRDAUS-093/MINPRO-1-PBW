# Penjelasan Code Setiap Fitur
1. Struktur Dasar HTML
   Pada bagian awal kode digunakan struktur dasar HTML yang terdiri dari:

   1. <html> sebagai elemen utama halaman.
    
   2. <head> berisi judul halaman dan link ke file CSS.
    
   3. <body> berisi seluruh tampilan website seperti navbar dan section.

   Struktur ini wajib digunakan agar halaman web dapat ditampilkan dengan benar di browser.

2. Navbar
   Navbar dibuat menggunakan tag <nav> yang berfungsi sebagai menu navigasi pada website.
   Di dalamnya terdapat:
   1. Judul website menggunakan <h2>
   
   2. Menu navigasi menggunakan <a> yang mengarah ke setiap section seperti Home, About, dan Certificates.

3. Section Home (Hero Section)
   Section Home dibuat menggunakan <section id="home">.
   Fitur yang terdapat di dalamnya:
   1. Judul perkenalan menggunakan <h1>
   
   2. Deskripsi singkat menggunakan <p>
   
   3. Gambar profil menggunakan <img>
   
   Bagian ini berfungsi sebagai tampilan awal yang memperkenalkan pemilik portfolio.

4. Section Tentang Saya, tentang diri pemilik website.
   Isi dari section ini meliputi:
   1. Deskripsi diri menggunakan paragraf <p>
   
   2. Skills yang ditampilkan dalam bentuk progress bar menggunakan <div>
   
   3. Daftar pengalaman menggunakan list 

5. Section Sertifikat
   Section Certificates menampilkan daftar sertifikat yang dimiliki.
   Struktur yang digunakan:
   1. Container utama menggunakan <div>
   2. Setiap sertifikat ditampilkan dalam bentuk card
   3. Card berisi gambar sertifikat dan judul
      
   Layout menggunakan CSS Flexbox atau Grid agar tampilan tersusun rapi.

6. Styling Menggunakan CSS
   File style.css digunakan untuk mengatur tampilan website seperti:
   1. Warna background
   2. Ukuran font
   3. Layout section
   4. Navbar styling
   5. Progress bar
   6. Card sertifikat
   7. Responsive design
      
   CSS membuat tampilan website menjadi lebih menarik dan rapi.

   # Teknologi
   Teknologi yang digunakan dalam pembuatan website ini adalah HTML sebagai struktur halaman dan CSS sebagai pengatur tampilan dan desain website.
